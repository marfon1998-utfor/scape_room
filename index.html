📊 ANÁLISE COMBINATÓRIA

Este jogo foi desenvolvido para trabalhar os conceitos fundamentais de Análise Combinatória, uma área da matemática que estuda a contagem, arranjos e combinações de elementos.

Conceitos e Fórmulas Utilizadas:

  • Princípio Fundamental da Contagem (P.F.C.):
    Se um evento ocorre de m formas e outro de n formas, então o total de combinações é: m × n
    Multiplicação de possibilidades independentes
  • Permutações: Ordenações usando TODOS os elementos
    P(n) = n! = n × (n-1) × (n-2) × ... × 1
    Ex: Organizar 4 livros em uma prateleira
  • Arranjos: Seleção ORDENADA de alguns elementos
    A(n, p) = n! / (n-p)!
    Escolher e ordenar p elementos de n disponíveis. A ordem importa!
  • Combinações: Seleção NÃO-ORDENADA de elementos
    C(n, p) = n! / (p! × (n-p)!)
    Escolher p elementos de n disponíveis. A ordem NÃO importa!
  • Permutações com Repetição: Ordenações com elementos repetidos
    P(n; n₁, n₂, ...) = n! / (n₁! × n₂! × ...)
    Onde n₁, n₂ são as frequências de cada elemento repetido

Vocês invadiram o escritório de um hacker internacional que sequestrou dados confidenciais de milhares de pessoas. Ao cruzarem a porta, o sistema de segurança foi ativado: as portas trancaram automaticamente.

Os dados estão armazenados em um drive físico protegido por um Cofre Criptográfico.

Para abrir o cofre e escapar, vocês precisam inserir uma combinação de 5 códigos numéricos. Cada código corresponde à resposta de uma pista deixada pelo próprio hacker na sala.

⚠️ ATENÇÃO ÀS REGRAS DO SISTEMA:

1. O cofre só valida os 5 códigos enviados de uma única vez.

2. Vocês têm apenas 3 CHANCES de input. Se errarem o bloco de senhas pela terceira vez, o sistema destruirá os dados e travará a sala definitivamente.
